Thank you for using ADVANCED BOOST Tools+. This CROME plugin allows you too add, edit, and customize boost options to P30, and P72 related ROMS. This plugin also includes Final Value Fuel Tools, and Load Based Closed Loop Disable plugins. The 'FV Fuel Tools' plugin is an advanced fuel control plugin to help compensate for necessary fuel changes that usually happen when adding boost and larger injectors. With FV Fuel Tools, all multipliers are relative to your final multiplier - if you want your O2, Postfuel, TPS multipliers to be the same as your overall multiplier, make sure they are set to 1.0!!! The 'Load Based Closed Loop Disable' plugin disables closed loop above a user defined load point.
