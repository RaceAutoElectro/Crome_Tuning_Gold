ADVANCED BOOST TOOLS+ 2.0


Advanced Boost Tools+ is a CROME plugin which allows the user to add boost capabilities to their stock ROMS. [Adv. BT+] is based on John Cui's original [BT+] boost code, but has more user defined options.

THIS VERSION DOES NOT INCLUDE A FUEL MULTIPLIER!!!
***If you are using injectors other than stock, manually run FuelTools.***
